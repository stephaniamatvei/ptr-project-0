## Running
To run a program, execute the following command in the program's file location:
```
$ elixir file_name.ex
```
Example:
```
$ cd /Users/stephaniamatvei/ptr_lab_1/lib/hello_ptr.ex
$ elixir hello_ptr.ex
```

## Testing
For testing, execute the following command in the project root:
```
$ mix test
```
