## Task
Write an application that would implement a Star Wars-themed RESTful API. The API should implement the following HTTP methods: <br /> 

• GET /movies <br />
• GET /movies/:id <br />
• POST /movies <br />
• PUT /movies/:id <br />
• PATCH /movies/:id <br />
• DELETE /movies/:id <br />

Use a database to persist your data.

## Database
Erlang Term Storage (ETS)

## How to run? 🏃‍♀️
```
mix run --no-halt
```
